
# is-reach

### 業務の流れ（3ステップ）
**スクリーニング**
従業員数・業種・地域や、求人・技術ブログなどの「公開シグナル」で企業を条件検索し、見込みリスト（だいたい 100〜500 社）を作る。ここは事前収集＋ルールベースが中心で、LLM はほぼ使わない。

**深掘り**
リストから人が選んだ企業だけ、HP・公開記事などをオンデマンドで収集し、LLM で分析。事業サマリ・推定課題・自社との接点・根拠 URL 付きの企業調書（ドシエ）を作る。

**メッセージ生成**
テナントが用意したテンプレ（自社紹介・CTA・トーン）に、ドシエ由来のパーソナライズ部分だけを LLM で埋めて文面案を出す。編集・コピーして人が問い合わせフォームに貼る。ステータス（未着手 / 送信済みなど）はツール上で管理。
